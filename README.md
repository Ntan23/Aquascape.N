# Aquascape.N 

A modular aquarium simulation built in Unity 6000.3.12f1, featuring dynamic fish AI, interactive environment objects, and automated configuration management.

## 🌟 Key Features

* **Dynamic Fish AI:** Fish utilize a raycasting system for obstacle avoidance and wall detection.
* **Self-Healing Config System:** Automatically generates a `config.json` and a `Spawnables` directory if missing, ensuring the app is always "Ready-to-Run."
* **Real-time File Watcher:** Detects changes in the `Spawnables` folder and `config.json` files.
* **Interactive Controls:** Supports spawning food, scaring fish, floating trash, and full camera control (Pan/Zoom).

## 🎮 Controls

| Input | Action |
| :--- | :--- |
| **LMB** | Spawn Food / Scare Fish / Float Trash |
| **MMB** | Camera Panning |
| **Scroll** | Camera Zoom (In / Out) |

## 🛠️ Technical Highlights

* **Dynamic Sprite Loading:** Implemented a system to load textures directly from external folders, allowing for easy asset customization without rebuilding.
* **Performance:** Implemented Object Pooling for spawnables (Fish, Trash and Food).
